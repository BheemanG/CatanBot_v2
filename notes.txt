Lumber(1), Brick(2), Wool(3), Grain(4), Ore(5)

Vertices (settlements/cities) - 54
Edges (roads) - 72
Tiles - 19

in_sequence: message count for WS IN, starting at 1, += 1 every message
out_sequence: message count for WS OUT, starting at 1, += 1 every message
id: game id (consistent all game) ex: "130"



WS IN (sequence always contains incrementing in_sequence):
    Type 1 (game settings)
        payload: dict of settings, see json
    Type 4 (initialization)
        see example json
    Type 6 (sequence: 1)
        payload: false?
    Type 28 (idk, appears after type31)
        payload: [] (empty)
    Type 30 (means a setlement vertex has been confirmed?)
        payload: [] (empty)
    Type 31: (available vertexes starter settlement can place road)
        payload: list of edges
    Type 78 (idk, appears after initialization) 
        "isActive": true/false
        "hasUsedDisableRequest": true/false

WS OUT (sequence always contains incrementing out_sequence):
    Action 8 (Discard):
        payload: card #s for discarded (1, 1, 1) = 3 lumber
    Action 11 (Place Road after initial settlement)
        payload: single number, represetnting the edge? (ex: 71)
    Action 15 (Place Initial Settlement)
        payload: vertex # (ex: 48)
    Action 66 (hover/select to place a road or settlement off initial settlement)
        payload: single number, represetnting the edge or vertex (ex: 71) or null if canceling
